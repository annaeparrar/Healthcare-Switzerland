# Healthcare-Switzerland
This project delivers an end-to-end analytics solution for a healthcare organization in Switzerland, focused on monitoring medical attendances and revenue generation (CHF).
Project Title: Healthcare Revenue & Attendance Analytics Dashboard – Switzerland

Project Overview
This project delivers an end-to-end analytics solution for a healthcare organization in Switzerland, focused on monitoring medical attendances and revenue generation (CHF) across key service segments:
•	Emergency
•	Hospitalization
•	Surgeries
•	Laboratory
The dashboard enables a monthly comparison between actual revenue and projected targets, providing immediate visibility into financial performance.

Business Objective
•	Analyze medical attendances and revenue by segment
•	Compare actual vs estimated revenue monthly
•	Identify overperformance and underperformance at a glance
•	Support financial and operational decision-making

Solution Architecture
•	SQL
o	Data extraction and aggregation
o	Business logic implementation
•	Power Query
o	Data cleansing and transformation
•	Power BI
o	Star schema data model
o	DAX measures for financial KPIs
o	Executive dashboard with conditional formatting

Data Model
•	Star schema design
•	Fact table with financial and operational metrics
•	Dimensions:
o	Date
o	Medical Service
o	Performance Status
Optimized for performance and scalability.

Dashboard Features
•	KPI cards for executive overview
•	Real vs Estimated revenue comparison
•	Conditional formatting (green/red performance indicators)
•	Monthly trend analysis of attendances
•	Interactive filters for time and service segmentation

Business Impact
•	Faster identification of financial deviations
•	Clear separation between volume and revenue issues
•	Reduced manual reporting effort
•	Reliable and consistent data for decision-makers

Tools & Technologies
•	SQL
•	Power BI
•	Power Query
•	DAX
•	Dimensional Modeling

Transforming clinical and financial data into actionable insights for healthcare decision-makers.
